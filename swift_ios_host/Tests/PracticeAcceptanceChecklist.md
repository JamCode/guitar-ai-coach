# Practice（练习 Tab）验收清单（Swift）

## 懒加载与健壮性
- 冷启动不进入「练习」Tab 时，不触发练习数据读取/解析（避免数据损坏导致启动崩溃）。
- 练习本地数据为损坏 JSON 时：进入「练习」Tab 不闪退，展示错误态并可重试（重试后回落空数据）。

## 首页
- 今日目标卡：目标 20 分钟、已完成分钟数/次数/连续天数显示正确，进度条随数据变化。
- 今日任务：三项任务均可进入子页。
- [x] 我的谱：可进入真实曲谱模块，支持相册多选/拍照导入、命名保存、删除与预览。
- [x] 详情页停留 ≥30 秒后，会写入一条「谱面跟练（sheet-practice）」练习记录。
- [x] 可执行 OCR 识别并进入校对页保存结构化内容；支持一键变调预览。
- 练习历史：空态文案正确；有数据时展示最近 3 条，且「查看全部」可进入全量列表。

## 通用计时练习（爬格子热身）
- 开始/暂停/结束：计时正常跳动，显示 `mm:ss`。
- 未开始直接点结束：提示“先开始练习再结束哦”。
- 返回拦截：点击返回会弹“离开练习？”确认。
- 结束弹窗：可选难度（1-5）、备注可选；保存后弹“记录已保存”并返回。

## 和弦切换
- 选择页：可切换调性、复杂度、和弦进行；预览和弦名随选择变化。
- 会话页：可开始/暂停/结束；点击和弦名弹离线指法 sheet；保存后历史出现对应记录（含 progressionId/musicKey/complexity）。

## 节奏扫弦
- 可选择节奏型；网格显示 8 格（1 & 2 & 3 & 4 &）。
- “图示说明”内容正确。
- 完成练习可保存记录，历史中显示节奏型名（由 rhythmPatternId 映射）。
