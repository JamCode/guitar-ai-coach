Wanle Guitar — Support

Last updated: 2026-04-23

This page provides technical support information for the Wanle Guitar iOS app, aligned with the App Store listing. We do not run a separate support server; the page is hosted on GitHub Pages and exists to help you get in touch and to provide basic self-service information.

1. App and platform

  • App name — 玩乐吉他 (Wanle Guitar)
  • Platform — iOS (see the App Store for minimum / compatible OS versions)
  • Overview — practice & history, ear training, local scores, transcription-style analysis, tuner / fretboard / chord tools, and more. Most processing is done on the device.

2. How to get help

You can contact the developers as follows:

  • In the app — go to Tools → Help & feedback and send a description using the system mail UI (a mail account must be configured on the device).
  • Email — write to 23766856@qq.com. We read messages when we can and try to respond, but we cannot guarantee real-time responses or a fixed SLA.

3. What to include when reporting a problem

To help us investigate, please include as much of the following as you can in your email or in-app report:

  • Device model, iOS version, and app version
  • What you saw and the steps to reproduce (roughly)
  • Whether microphone / photos / file access is involved, and whether permission was granted at the time

4. Data and privacy

For data collection, permission purposes, and deletion, see the Wanle Guitar Privacy Policy.

This page is not a legal or contractual warranty. If you have issues with account security, payments, or system-level errors, also consult Apple’s official support channels where appropriate.

玩乐吉他技术支持

更新时间：2026-04-23

本页面为「玩乐吉他」iOS 应用的技术支持信息，与 App Store 展示一致。我们未提供独立服务器，本页托管于 GitHub Pages，仅用于帮助用户联系与自助说明。

1. 应用与平台

  • 应用名称：玩乐吉他
  • 平台：iOS（具体兼容版本以 App Store 页面为准）
  • 功能概览：练习与记录、视唱练耳、本地曲谱、扒歌分析、调音/指板/和弦工具等，主要能力在设备本地处理。

2. 如何获取帮助

你可以通过以下方式联系开发者：

  • 应用内：打开「工具」→「帮助与反馈」，使用系统邮件发送问题描述（需本机已配置邮件账户）。
  • 邮件：直接发送邮件至 23766856@qq.com。我们会在方便时阅读并尽量回复，但无法承诺实时响应或 SLA。

3. 反馈问题时请尽量附带

为便于定位问题，建议在邮件/反馈正文中写清（能写多少写多少）：

  • 设备型号、iOS 系统版本、App 版本号；
  • 问题现象与大致发生步骤；
  • 是否涉及麦克风/相册/文件访问、以及当时权限是否已允许。

4. 数据与隐私

关于数据收集、权限用途与删除方式，请参阅 《玩乐吉他隐私政策》。

本页面不构成法律或合同承诺。若你遇到涉及账号安全、支付或系统级异常，请同时参考 Apple 官方支持渠道。
