# Guitar AI Coach

吉他和弦进行生成与指法说明（前端 Vue + 后端 Python，部署说明见 `deploy/README.md`）。

## 与 AI 协作开发约定

在通过 Cursor 等工具改代码时，建议遵守以下流程，避免未对齐就动手实现：

1. **先对齐方案**：由 AI 说明目标、可选实现方式、利弊与推荐选项；必要时列出将改动的文件范围。
2. **再开始开发**：**只有你明确同意之后**（例如回复「可以开发了」或指定某一方案）再编写、修改代码与执行部署。
3. **仅要分析时**：若你暂时不需要改仓库，请直接说明，AI 应只给结论与建议而不提交代码变更。

以上约定对人与 AI 均适用，便于控制范围与审查成本。
