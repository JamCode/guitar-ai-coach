练习 Tab：页面与点击去向

依据 swift_ios_host 中「练习」Tab（PracticeHomeView / PracticeLandingView.swift）与 swift_app 包内视唱练耳视图梳理。流程图中实线为导航栈 push，虚线为 Sheet 或 Alert。 下方「UI 线框」为浏览器内绘制的低保真示意（非截图），用于对齐信息结构与主要控件位置。 首次进入练习 Tab 时先显示占位页，切换一次后即挂载真实首页（见 SwiftEarHostApp.swift）。

实线：导航栈 push（NavigationLink） 虚线：Sheet 弹出层或 Alert（图中统一用虚线表示非 push 交互）
